
# nvs 的学习使用

## Introduction

windows 上可以用 nvs 来进行 node 的版本切换。        

[gitub 地址](https://github.com/jasongin/nvs)         

选择合适的版本的 .msi 文件安装即可。        

## Usage

这里仅理列举一些常用的命令，方便自己以后查阅。          

|命令|作用|
|---|---|
|nvs -v|查看当前 nvs 版本|
|nvs add node/12.13.0|下载12.13.0的node|
|nvs use node/12.13.0/x64|切换到12.13.0版本的node|
|nvs ls|可以查看当前有下载的所有版本吧的 node |
