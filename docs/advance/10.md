## 手写 instanceof


 运算符不会写，写 instanceof 函数

 instanceof 作用。 

 判断对象是否在构造函数的原型链上。 

```js
function myinstanceof(obj, constructor) {
    let pro = constructor.prototype;
    let p = obj.__proto__;
    while(p !== null) {
        if(p === pro) {
            return true;
        } else {
            p = p.__proto__;
        }
    }
    return false;
}
```
